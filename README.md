# git
## Трэк удалённой ветки
```bash
git branch --track test origin/test
```
## Сбросить мердж
```bash
git reset --merge
```
## Показать удалённые ветки
```bash
git branch -r
git branch -a
git branch -vv
```
## Удаление локальной ветки
```bash
   git branch -d ci
```
```bash
   git branch -D ci
```
